### Kết quả đạt được
* Hiểu rõ khái niệm điện toán đám mây và kiến trúc AWS. 
* Nắm được các nhóm dịch vụ chính: 
    * Compute (EC2)
    * Networking (VPC)
    * Database (RDS)
* Tạo và cấu hình thành công tài khoản AWS Free Tier. 
* Làm quen với AWS Management Console: 
    * Cách tìm kiếm dịch vụ 
    * Quản lý tài nguyên 
    * Hiểu giao diện hệ thống 
* Cài đặt và cấu hình AWS CLI: 
    * Thiết lập Access Key và Secret Key 
    * Cấu hình region mặc định 
    * Kiểm tra kết nối CLI 
* Thực hiện các thao tác cơ bản bằng AWS CLI: 
    * Kiểm tra thông tin tài khoản 
    * Liệt kê region 
    * Xem thông tin EC2 
    * Quản lý key pair 
* Triển khai thành công EC2: 
    * Cấu hình Security Group 
    * SSH vào server 
    * Gắn và kiểm tra EBS 
* Có khả năng sử dụng kết hợp AWS Console và CLI để quản lý hệ thống cloud.
